Descrição do programa e suas funcionalidades:

O programa simula o aplicativo que será desenvolvido para celular. Nele, será possível verificar informações da(s) lixeira(s) do usuário a partir de dicionários (e valores simulados com a biblioteca random do Python de sensores da lixeira), verificar se possui notificações a respeito da lixeira inteligente e tirar dúvidas sobre como utilizar, como funiona a lixeira inteligente e como funciona o sistema de coleta do lixo.

O programa contém um Menu Principal feito com a estrutura de repetição while com um print na tela com opções que o usuário pode escolher a a partir do input e atribuição de variável. A partir dessa variável, o programa analisa com operadores de comparação "if" e "elif" a resposta do usuário e redireciona para a seção escolhida.

Se escolher a opção "[1] Notificações", o aplicativo mostraria se ele possui alguma notificação, por exemplo se o caminhão da coleta está a caminho ou se a coleta já foi realizada. Além disso, há um print de um pequeno menu mostrando que se o usuário digitar 888 ou 999 em qualquer input, ele volta para o menu principal ou vai para o menu de confirmação de saída, respectivamente. A análise é feita através da função continuar() e se o input for diferente dessas duas opções, há um print de uma mensagem de erro e o usuário fica no loop até digitar uma opção válida. Após isso, o usuário é redirecionado para o menu escolhido.

Se caso escolher a opção "[2] Preciso de ajuda", o aplicativo o redirecionaria para o "Menu Ajuda" onde seria mostrado novamente um menu, novamente feito com a estrutura de repetição while, mas dessa vez com exemplos de dúvidas e uma opção para entrar em contato direto com o suporte, em que é printado o email da equipe de suporte da nossa empresa e o usuário poderia enviar um email para falar diretamente com o suporte. A escolha é feita pelo usuário, atribuída a uma variável e através dos operadores de comparação "if", "elif" e "else" é verificado qual foi a dúvida e há um print na tela com a resposta da dúvida. Caso o input do usuário não for nenhuma das opções fornecidas, há um print de uma mensagem de erro e o usuário fica no loop até digitar uma opção válida. Após isso, o usuário é redirecionado para o Menu Ajuda novamente.

Se escolher a opção "[3] Enviar sugestões", o aplicativo o redirecionaria para a seção "Sugestões" onde o usuário poderia relatar alguma insatisfação, algum problema ou dar uma sugestão para o sistema da lixeira inteligente ou até mesmo para o aplicativo. Após o input do usuário, é feita uma verificação do input no programa principal para ver se é válido dentro das opções válidas (888, 999 ou qualquer texto).

Se escolher a opçao "[4] Verificar Lixeiras", o aplicativo mostraria as lixeiras do usuário com uma opção para escolher qual ele deseja ver os detalhes. Após escolher, passa por um if verificando a escolha do usuário e caso esteja certa, mostra as informações do dicionário da lixeira escolhida com um laço for com a chave e valor do dicionário "lixeiras". Depois, o usuário volta para o menu de Lixeiras.

Se o usuário escolher a opção "[999] Sair do aplicativo" em qualqueur input, ele será redirecionado para o Menu Operações Realizadas (dentro de um while) e terá as suas operações realizadas no aplicativo listadas em ordem que ocorreram. Ele terá a opção de '[1] Voltar' ou '[2] Sair'. Caso o input for diferente de 1 ou 2, será exibida uma mensagem de erro. Caso for igual, o usuário sai do loop, e de acordo com a resposta dele, o while principal que engloba tanto o programa principal quanto a confirmação de saída se encerrará (ligar = 0) e será printada uma mensagem avisando que o aplicativo está sendo fechado ou o usuário voltará para o while do programa principal (fechar = 1).

Além disso, um pequeno menu semelhante ao pequeno menu citado em notificações, porém é um menu formatado dentro da função menuSairVoltar(). Nela, é verificado se o menu é o principal e, se for, somente será printado a opção de sair do aplicativo (999). Senão, também será printado a opção para voltar ao menu principal (888). Essa função é chamada antes de todos os inputs, com exceção do input final no Menu Operações Realizadas.

As outras funções presentes no programa são para a formatação de alguns prints para deixar o programa mais bonito visualmente e mais fácil de ser compreendido. A função linSimples(tam) define um print do traço "-" vezes o parâmetro tam. A função linDetalhe() tem a mesma funcionalidade que a linSimples(), porém ao invés de ser apenas printado diversos "-", é printado "-=", para diferenciar da outra linha. A função subli(frase, variavel) tem como funcionalidade printar "-" embaixo de um texto (frase) somado com o input do usuário respectivo dessa frase, com o objetivo dividir a opção escolhida pelo usuário do programa que virá posteriormente. Já as funçãos voltandoMenu(menu) e carregandoMenu(menu), tem como objetivo exibir que o usuário está sendo direcionado para algum menu, de modo com que fique mais fácil de saber o caminho que foi feito pelo próprio usuário.
